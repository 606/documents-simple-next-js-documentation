## Overview

Jan is an open-source ChatGPT alternative that runs 100% offline, supporting multiple models including Llama and Mistral.

## Key Features

- **100% Offline**: No internet connection required
- **Open Source**: AGPLv3 licensed
- **Model Support**: Llama, Mistral, and other open-source models
- **Cross-Platform**: Windows, macOS, Linux support
- **Extensions**: Plugin system for additional features

## Installation

Download from [jan.ai](https://jan.ai)

## Usage

1. **Download Models**: Use the built-in model downloader
2. **Start Chatting**: Select a model and begin conversation
3. **Customize**: Adjust parameters and settings

## Supported Models

- Llama 2/3 series
- Mistral 7B
- Other open-source chat models
